# schemata
A Visual Table Designer

- Single Page Application
- Created fully by Claude.ai
- Tools to introduce database to a spreadsheet minded friend
- Tabs: Spreadsheet, Table Designer, Data Viewer (CRUD), SQL Query Builder
- Export and Import json file (for table design and data)
- Export tables create command as sql (for SQLite)
